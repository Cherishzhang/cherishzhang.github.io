c++关键字之友元(friend)

类的机制实现了数据的隐藏与封装，类的数据成员一般为私有成员，成员函数一般公有，作为类与外界间的通信接口。友元机制允许类的非公有成员被一个类或函数访问，包括友元函数和友元类。友元函数与友元类不是类的一部分，友元提高了程序的运行效率(减少了类型检查和安全检查所需的时间开销)，破坏了类的封装性和隐藏性。

友元的特性： 1. 不具有相互性，只有单向性 类B是类A的友元，类A不一定是类B的友元，具体要看是否有相应的声明 2. 不能被继承 B是A的友元，C是B的子类，不能得出C是A的友元 3. 不具有传递性 B是A的友元，C是B的友元，不能得出C是A的友元 4. 声明一个友元函数或者友元类，就是把自己完全暴露给对方

  • 友元函数 友元函数是可以直接访问类的私有成员的非成员函数。它是定义在类外的普通函数，它不属于任何类，但需要在类的定义中加以声明，声明时只需在友元的名称前加上关键字friend，其格式如下： friend 类型 函数名(形式参数);
    • 友元函数的声明可以类中的任何地方(public,private,protected)
    • 一个函数可以是多个类的友元函数，只需要在各个类中分别声明。友元函数的调用与一般函数的调用方式和原理一致。

普通函数是类(1个或多个)的友元函数

#include<iostream>
using namespace std;
class MyClass2;
class MyClass
{
  public:
    MyClass(string name):m_name(name){}
    friend void Display(MyClass &myclass,MyClass2 &myclass2); //声明一个友元函数；MyClass可是非常信任Display呢
  protected:
    string m_name;
};
class MyClass2
{
  public:
    MyClass2(string name):m_name(name){}
    friend void Display(MyClass &myclass, MyClass2 &myclass2);
  protected:
    string m_name;
};
void Display(MyClass &myclass, MyClass2 &myclass2) //定义友元函数，不需要加MyClass::
{
  cout << "Access Protected data:" << myclass.m_name << endl;
  cout << "Access Protected data from 2:" << myclass2.m_name << endl;
}
int main()
{
  MyClass test("Class A");
  MyClass2 test2("Class A2");
  Display(test,test2);
  return 0;
}

类的成员函数可以是另一个类的友元

#include<iostream>
using namespace std;
class MyClass2;
class MyClass{
public:
  MyClass(string name):m_name(name){}
  void Function(MyClass2 &myclass2);
private:
  string m_name;
};

class MyClass2{
public:
  MyClass2(string name):m_name(name){}
  friend void MyClass::Function(MyClass2 &myclass2); //友元函数声明
private:
  string m_name;
};
void MyClass::Function(MyClass2 &myclass2){ //友元函数定义
  cout << "" << myclass2.m_name << endl;
}
int main(){
  MyClass test("Class A");
  MyClass2 test2("Class A2");
  test.Function(test2);
  return 0;
}
  • 友元类 友元类的所有成员函数都是另一个类的友元函数，都可以访问另一个类中的隐藏信息（包括私有成员和保护成员）。当希望一个类可以存取另一个类的私有成员时，可以将该类声明为另一类的友元类。

类是另一个类的友元

#include<iostream>
using namespace std;
class MyClass2;
class MyClass{
public:
  MyClass(string name):m_name(name){}
  friend class MyClass2; //友元类声明
private:
  string m_name;
};
class MyClass2{
public:
  MyClass2(string name):m_name(name){};
  void Display(MyClass &myclass);
private:
  string m_name;
};
void MyClass2::Display(MyClass &myclass){
  cout << myclass.m_name << endl; //访问A的私有成员
}
int main(){
  MyClass test("class A");
  MyClass2 test2("class A2");
  test2.Display(test);
  return 0;
}
