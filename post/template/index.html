Template in C++

对于重载(overloading)函数，c++的检查机制能够通过函数参数的不同及所属类的不同，正确的调用不同的版本。而模板可以解决重载定义不全面而可能带来的调用错误。

模板是实现代码重用机制的一种工具，可以实现类型参数化，即把类型定义为参数，从而实现真正的代码可重用性。模板分为两类，一是函数模板，而是类模板。

  1. 函数模板
#include<iostream>
using namespace std;
//声明一个用来比较两个相同数据类型的参数的大小，class也可以被typename代替
//T可以被任何字母或数字代替
template<class T> //or template<typename T>
T min(T x, T y)
{
  return (x < y)?x:y;
}
//重载函数
int min(int x, int y)
{
  return (x < y)?x:y;
}
int min(char x, char y)
{
  return (x < y)?x:y;
}
int main()
{
  int n1=2,n2=10;
  double d1=1.5,d2=5.6;
  cout<<"较小整数:" << min(n1,n2) << endl; //调用重载函数
  cout<<"较小实数:" << min(d1,d2) << endl; //无对应的重载函数，则调用模板
}

重载函数和模板一起使用时，先考虑重载函数，后考虑模板类，最后考虑类型转换。

  1. 类模板
#include<iostream> 
using namespace std;
template<class T>
class Base{
public:
  T a;
  Base(T b):a(b){} //多个参数时，用逗号连接
  T getA(){return a;} //类内定义
  void setA(T c); 
};
template <class T> //成员函数在类外定义,需要使用函数模板
void Base<T>::setA(T c){
  a = c;
}
int main(){
  Base <int> b(4);
  cout << b.getA() << endl;
  Base <double> bc(4);
  bc.setA(4.3);
  cout << bc.getA() << endl;
  return 0;
}
