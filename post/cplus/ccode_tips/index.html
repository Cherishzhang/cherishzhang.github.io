cmath

  • M_PI – 圆周率
  • sin – 余弦
  • asin – 反余弦
  • pow(i,j) – i的j次方,溢出考虑
  • 精度问题 – 判断两个浮点数大小关系，直接用=,<,>比较，可能出错。一般常见的解决方法是设定一个极小量epsilon(一般写作eps=1e-6)来辅助比较
abs(a-b) < eps //a==b
abs(a-b) > eps //a!=b
a-b > eps // a>b
a-b > -eps //a>=b
a-b < -eps //a<b
a-b < eps //a<=b

整型元素

  1. int类型的范围：-2^31 ~ 2^31-1 即\(2*10^9\)左右，long long类型的范围：-2^63 ~ 2^63-1 即\(9*10^{18}\)左右。如果会溢出，考虑long long类型。

  2. 左移右移(c语言特性): << 左移1位，相当于乘以2，有可能导致符号变化

字符与字符串

  1. const char, char const, char* const的区别：
    声明从右向左读：
    char* const cp; //cp is a const pointer to char
    const char *p 或 char const *p; //p is a pointer to const char

  2. c++标准规定，const关键字放在类型或变量名之前等价。
    const int n=5;// int const n=5;

  3. 如何判断字符指针的结束位置：’\0’

  4. 字符串字典序比较：

    strcmp(s1,s2); //s1和s2必须是char数组, 如果是string比较， 直接s1 == s2即可

  5. 字符转ascii码：

    char c = '1';
int ascii = c;

//关于queue的操作 - 先进先出 - 广度优先搜索 - #include <queue>
/*
 *两个模板参数，元素类型，容器类型，前者必须，后者默认deque
 */
queue<int> p;
p.push(sth); //入队列
p.pop(); //出队列
int front = p.front(); //访问头部元素
int back = p.back(); //访问尾部元素
p.empty(); //判断队列空
p.size(); //元素个数

//关于stack的操作 - 先进后出 - 深度优先搜索 - #include <stack>
/*
 *该模板类需要两个参数，元素类型，容器类型，前者是必须的。默认容器类型为deque.
 *
 */
stack<int> sta;
sta.push(5); //入栈
sta.push(10);
sta.push(15);
sta.pop(); //出栈，只删除元素，不返回元素
int x = sta.top(); //访问栈顶
bool empty = sta.empty(); //判断栈空
int size = sta.size(); //元素个数

/*priority_queue<Type, Container, Functional> - 优先队列 - #include<queue>
 *Type为数据类型，Container为保存数据的容器，Functional为元素比较方式
 *Container必须是用数组实现的容器，如vector,deque，但不能用list
 *STL默认容器是vector,比较方式默认用<,如果缺省后面两个参数，该优先队列是大顶堆头元素最大。*/
//声明大顶堆
priority_queue<int, vector<int>, less<int> >q;//equals to priority_queue<int> q1;
for(int i=0; i<10; i++)
    q.push(i);
while(!q.empty()){
    cout << q.top() << endl;
    q.pop();
}
//声明小顶堆
priority_queue<int,vector<int>,greater<int> >q;
